# SPEC Deepen Command

Interactively deepen an existing SPEC through analysis and focused deep-dive.

## Usage

```text
/gwt:gwt-spec-deepen [spec-id]
```

## Steps

1. Load `.claude/skills/gwt-spec-deepen/SKILL.md` and follow the workflow.
2. Read all SPEC artifacts (spec.md, plan.md, tasks.md).
3. Produce a deepening report with categorized points.
4. Deep-dive on user-selected points with targeted questions.
5. Update artifacts with confirmed changes.
