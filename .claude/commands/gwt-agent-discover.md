# Agent Discover Command

Use this command to list active agent panes in the current project.

## Usage

```text
/gwt:gwt-agent-discover [context]
```

## Steps

1. Load `.claude/skills/gwt-agent-discover/SKILL.md` and follow the workflow.
2. Run `pane list` to enumerate active panes.
3. Display pane IDs, agent types, branches, and statuses.
