Synapse Visual PoC

Neural Link Playground

TypeScript + Canvas で粒子ネットワークを描画し、シナプスをイメージしたパルスを表現します。 このページはプロダクションコードから独立した検証用スペースです。

Back to repo GitHub ↗

Design Notes

  • 粒子はブラウン運動のようにランダムドリフトし、壁で反射します。
  • 一定距離内のノードを結線し、電気信号をイメージしたパルスを流します。
  • TypeScript でノード／エッジのデータ構造を型安全に管理しています。
  • Canvas の低レベル API を直接制御し、軽量にアニメーションしています。
