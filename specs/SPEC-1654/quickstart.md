# Quickstart
1. Open a project and verify the shell shows `Agent Canvas` and `Branch Browser`
2. Confirm no Sidebar is present
3. Open `Branch Browser`, switch between `Local`, `Remote`, and `All`
4. Create or focus a worktree and verify a worktree tile appears on `Agent Canvas`
5. Launch an agent from that worktree and verify an agent tile is linked by an edge
6. Open a terminal from that worktree and verify a terminal tile is linked by an edge
7. Open a second window and confirm shell/canvas state remains window-local
