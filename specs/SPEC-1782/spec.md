# Quick Start（前回設定復元）

## Background

gwt-tui の Wizard に QuickStart ステップを追加し、ブランチごとの前回起動パラメータを復元してワンアクションでエージェントを起動できるようにする。起動パラメータは `~/.gwt/quick-start/` に保存し、最近の 5 件を表示する。

## User Stories

### US-1: 前回と同じ設定でエージェントを即座に起動する

開発者として、Wizard の最初のステップで前回の起動設定を選択し、ワンアクションで同じ構成のエージェントを起動したい。

### US-2: 最近の起動履歴から選択する

開発者として、最近の 5 件の起動履歴から選択して、異なるブランチ・設定のエージェントを素早く再起動したい。

### US-3: QuickStart をスキップして新規設定する

開発者として、QuickStart をスキップして通常の Wizard ステップに進み、新しい設定でエージェントを起動する選択肢も持ちたい。

## Acceptance Scenarios

### AS-1: QuickStart ステップの表示

- Wizard を開始する
- 最初のステップとして QuickStart が表示される
- 最近の 5 件の起動履歴がリスト表示される

### AS-2: 前回設定での即座起動

- QuickStart で前回の起動設定を選択
- Enter で確定
- 前回と同じパラメータ（ブランチ、エージェントタイプ、引数等）でエージェントが起動される

### AS-3: ブランチごとの設定保存

- Wizard で設定を完了しエージェントを起動
- 起動パラメータが `~/.gwt/quick-start/{branch_name}.json` に保存される

### AS-4: 履歴なし時のスキップ

- 初回起動で履歴がない場合
- QuickStart ステップはスキップされ、通常の Wizard ステップに進む

### AS-5: 新規設定へのスキップ

- QuickStart で「新規設定」を選択
- 通常の Wizard ステップに進む

## Functional Requirements

| ID | 要件 |
|----|------|
| FR-001 | Wizard の最初のステップとして QuickStart を表示する |
| FR-002 | ブランチごとの起動パラメータを `~/.gwt/quick-start/` に JSON で保存する |
| FR-003 | 最近の 5 件の起動履歴を表示する |
| FR-004 | 選択した起動設定でエージェントを即座に起動する |
| FR-005 | 「新規設定」オプションで通常の Wizard ステップにスキップする |
| FR-006 | 履歴がない場合は QuickStart ステップをスキップする |
| FR-007 | 起動パラメータにはブランチ名、エージェントタイプ、コマンド引数、worktree パスを含む |

## Non-Functional Requirements

| ID | 要件 |
|----|------|
| NFR-001 | 起動履歴の読み込みは 100ms 以内に完了する |
| NFR-002 | 保存ファイルの破損時はスキップしてフォールバックする |
| NFR-003 | 古い履歴は自動的にプルーニングする（最大 50 件保持） |

## Success Criteria

| ID | 基準 |
|----|------|
| SC-001 | Wizard の最初のステップに QuickStart が表示される |
| SC-002 | 最近の 5 件の起動履歴が正しく表示される |
| SC-003 | 選択した設定でエージェントが正しく起動される |
| SC-004 | 起動パラメータが `~/.gwt/quick-start/` に保存される |
| SC-005 | 「新規設定」でスキップが動作する |
