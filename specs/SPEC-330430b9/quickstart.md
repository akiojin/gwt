# クイックスタート: セッション変換一覧の内容表示とSpaceプレビュー

## 手順

1. `cargo run` で gwt を起動する。
2. 対象ブランチを選択し Wizard を開く。
3. Execution Mode で `Convert` を選択。
4. セッション一覧でセッション名（Worktree名）・開始ユーザーメッセージ・更新日時が表示されることを確認。
5. Space を押してプレビューを開き、ヘッダーにセッション名（Worktree名）と先頭10メッセージが表示されることを確認。
6. Space または Esc でプレビューを閉じる。
7. セッションを確定すると「Converting session...」のスピナーが表示されることを確認。

## 期待結果

- 一覧にセッションID/メッセージ数は表示されない。
- セッション名（Worktree名）が無い場合は `No name` を表示。
- 開始ユーザーメッセージが無い場合は `No user message` を表示。
- 読み込み失敗時は `Unavailable` を表示。
