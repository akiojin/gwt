### Core service
- `IGitService`
  - worktree, branch, commit, diff, stash, cleanup, tag 系 API を提供
- `GitCommandRunner`
  - `RunAsync(args, workingDir, ct, timeoutMs) -> (stdout, stderr, exitCode)`
- `GitService`
  - `GitCommandRunner` を注入し、CLI 出力を parse して model 化
