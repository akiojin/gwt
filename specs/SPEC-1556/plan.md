### 技術コンテキスト

- Rust/Tauri 版の ~/.gwt/ データ構造（TOML/JSON）を C# で読み取り、Unity 版の JSON 形式に変換する
- TOML パーサーの C# ライブラリを選定・導入する
- 初回起動時に自動実行し、メインスレッドをブロックしない非同期処理で実装する

### 実装アプローチ

- 完全自動マイグレーション（ユーザー操作不要）
- 失敗時はスキップして新規状態で起動（エラーでブロックしない）
- 元ファイルは保持し、バックアップも作成する

### フェーズ分割

1. ~/.gwt/ の既存データ構造の棚卸し
2. 各ファイル形式の変換マッピングを定義する（agents.toml含む全TOML対象）
3. TOML パーサーの選定・導入
4. 各設定ファイルの変換ロジックを実装する（元ファイル保持）
5. セッションデータの変換ロジックを実装する
6. バックアップ機構を実装する（ロールバックUIは不要）
7. プログレスバー UI を実装する
8. テストを作成する
