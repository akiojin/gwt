# 要件チェックリスト: エラーポップアップ・ログ出力システム

**目的**: SPEC-e66acf66 の機能要件が正しく仕様化されているかを検証する
**作成日**: 2026-01-22
**機能**: [spec.md](../spec.md)

## エラーポップアップ表示

- [ ] CHK001 エラー発生時にポップアップが画面中央に表示される
- [ ] CHK002 ポップアップ幅が最大70文字に制限される
- [ ] CHK003 エラーコード（GWT-EXXX形式）がタイトルに表示される
- [ ] CHK004 エラーメッセージが本文に表示される
- [ ] CHK005 詳細セクションが表示される（スクロール可能）
- [ ] CHK006 サジェスチョンセクションが表示される

## キーボード操作

- [ ] CHK007 [Enter]キーでポップアップが閉じる
- [ ] CHK008 [Esc]キーでポップアップが閉じる
- [ ] CHK009 [l]キーでログ画面に遷移する
- [ ] CHK010 [c]キーでエラー情報がクリップボードにコピーされる

## マウス操作

- [ ] CHK011 ポップアップ外のクリックでポップアップが閉じる
- [ ] CHK012 フッターの[l]クリックでログ画面に遷移する
- [ ] CHK013 フッターの[c]クリックでクリップボードにコピーされる
- [ ] CHK014 詳細セクションがマウスホイールでスクロールする

## エラーキュー

- [ ] CHK015 複数エラーがキューに追加される
- [ ] CHK016 キュー位置がタイトルに表示される（例: Error (1/3)）
- [ ] CHK017 ポップアップを閉じると次のエラーが表示される

## ログ出力

- [ ] CHK018 エラーがログファイルに記録される
- [ ] CHK019 ログにエラーコードが含まれる
- [ ] CHK020 ログにカテゴリが含まれる
- [ ] CHK021 ログにメッセージが含まれる

## 重大度表示

- [ ] CHK022 Error（赤）の色分けが正しい
- [ ] CHK023 Warning（黄）の色分けが正しい
- [ ] CHK024 Info（青）の色分けが正しい

## 範囲外の確認

- [ ] CHK025 セッションサマリー取得失敗はポップアップ表示しない（セッションタブに維持）
