- Rust
  - `mergeStateStatus=BLOCKED` + required check `queued` => `checking`
  - `mergeStateStatus=BLOCKED` + required check `in_progress` => `checking`
  - required check failure => `blocked`
  - `CHANGES_REQUESTED` => `blocked`
- Svelte
  - `PrStatusSection` で pending required checks が `Checking merge status...`
  - `PrStatusSection` で required check failure が `Blocked`
  - Sidebar badge class が pending-check case で `checking`
