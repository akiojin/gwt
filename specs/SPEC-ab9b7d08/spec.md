# 機能仕様: Aboutダイアログにバージョン表示 + タイトルにプロジェクトパス表示（GUI）

**仕様ID**: `SPEC-ab9b7d08`  
**作成日**: 2026-02-10  
**ステータス**: 実装完了  
**カテゴリ**: GUI  
**入力**: ユーザー説明: "Aboutでバージョン表示。Windowタイトルはプロジェクトパスを表示"

## 背景

現在の gwt GUI は、ウィンドウタイトルと About の情報設計を調整中であり、
バージョン表示の場所を明確にする必要がある。
ユーザー要望として、**ウィンドウタイトルはプロジェクト識別に集中**し、
**バージョンは About ダイアログで確認**できる形が求められる。

よって、タイトルは `gwt` / `gwt - <projectPath>` に固定しつつ、
Tauri 実行環境で取得できるアプリバージョンを About ダイアログへ表示する。

## ユーザーシナリオとテスト *(必須)*

### ユーザーストーリー 1 - タイトルでプロジェクトパスを確認できる (優先度: P1)

開発者として、複数プロジェクト運用時にタイトルバーだけで対象プロジェクトを判別したい。

**受け入れシナリオ**:

1. **前提条件** gwt GUI を起動し、まだプロジェクト未選択、**操作** タイトルバーを確認する、**期待結果** タイトルが `gwt` になる
2. **前提条件** Open Project でプロジェクトを開く、**操作** タイトルバーを確認する、**期待結果** タイトルが `gwt - <projectPath>` 形式になる

### ユーザーストーリー 2 - Aboutダイアログでバージョンを確認できる (優先度: P1)

開発者として、実行中バージョンを About ダイアログから確認したい。

**受け入れシナリオ**:

1. **前提条件** Tauri API が利用可能な環境、**操作** About ダイアログを開く、**期待結果** `Version <version>` が表示される
2. **前提条件** バージョン取得に失敗する環境、**操作** About ダイアログを開く、**期待結果** 例外が発生せず `Version unknown` が表示される

### エッジケース

- バージョン取得が失敗した場合でも UI 初期表示はブロックされず、About は `Version unknown` と表示すること
- タイトルおよび About の表示文言は英語のみで構成されること

## 要件

### 機能要件

- **FR-460**: システムは、プロジェクト未選択時のウィンドウタイトルを `gwt`、プロジェクト選択時を `gwt - <projectPath>` として表示しなければ**ならない**
- **FR-461**: システムは、About ダイアログに `Version <version>` を表示しなければ**ならない**
- **FR-462**: バージョンが取得できない場合、システムは例外を投げては**ならず**、About に `Version unknown` を表示しなければ**ならない**

### 非機能要件

- **NFR-460**: バージョン取得は best-effort とし、UI の初期表示をブロックしては**ならない**
