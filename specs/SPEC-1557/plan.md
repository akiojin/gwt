### 技術コンテキスト

- Unity 6 + VContainer（DI）+ UniTask（非同期）
- 既存 Rust/Tauri 版の 8 Tauri コマンドを C# サービスメソッドに移植
- 依存: #1542（データ永続化）、#1543（Git 操作）、#1546（スタジオ UI）

### 実装アプローチ

1. `IProjectLifecycleService` インターフェースを定義する
2. パスプローブ・Git リポジトリ検出を実装する
3. プロジェクト開閉のライフサイクル管理を実装する（初回表示5秒以内、重い処理はバックグラウンド）
4. bare リポジトリ作成・移行ジョブを実装する
5. アプリケーション終了シーケンス（全停止・永続化）を実装する
6. 終了確認ダイアログを実装する
7. VContainer DI 登録・テスト

### フェーズ分割

```
Phase S: セットアップ
  └─ インターフェース定義、データ型定義

Phase F: 基盤
  └─ パスプローブ、Git リポジトリ検出、プロジェクト情報取得

Phase U: ユーザーストーリー実装
  └─ プロジェクト開閉、bare リポ作成・移行、終了シーケンス、確認ダイアログ
  └─ 段階的ロード（初回表示5秒以内 + バックグラウンド後追い）

Phase FIN: 最終化
  └─ VContainer DI 登録、統合テスト、パフォーマンステスト
```
