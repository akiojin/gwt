1. Unity Hub で Unity 6000.3.8f1 をインストール
2. `gwt/gwt/` ディレクトリを Unity Editor で開く
3. Package Manager で VContainer, UniTask を確認（既にインポート済み）
4. Play ボタンでスタジオシーンを起動
5. コンソールログ（`~/Library/Logs/Unity/Editor.log`）でエラーがないことを確認
6. EditMode テスト実行: Unity Editor > Window > General > Test Runner > EditMode > Run All
7. PlayMode テスト実行: Unity Editor > Window > General > Test Runner > PlayMode > Run All
