# Quickstart
1. Open the Branch Browser shell surface that consumes `#1644`
2. Switch between `Local`, `Remote`, and `All`
3. Select a remote-only ref and verify the domain offers worktree creation rather than existing-worktree focus
4. Select a local branch with an existing worktree and verify the domain resolves to that worktree instance
5. Inspect a worktree instance and verify display name, linkage, divergence, and safety information are available from the same projection
6. Classify a branch inventory/performance/cache/invalidation change and verify it belongs to `#1644`
7. Classify a project-open orchestration change as `#1647` and an issue-cache sync change as `#1714`
