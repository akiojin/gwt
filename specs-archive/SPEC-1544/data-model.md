### Core service
- `IGitHubService`
  - auth / issues / PR / CI checks / crash report を扱う
- `GhCommandRunner`
  - `gh` CLI 実行ラッパー
  - `RunAsync(args, workingDir, ct, timeoutMs)`
  - `RunJsonAsync(...)`
- `GitHubService`
  - `GhCommandRunner` 注入
  - gh JSON を deserialize して model へ変換

  - `Color`
- `GitHubAssignee`

- `PullRequest`
  - `Number`
  - `Title`

  - `State`
  - `Url`

- `WorkflowRunInfo`

### Internal DTOs
- gh CLI JSON 用 DTO を service 内 private/internal 型で保持
- deserialize policy は camelCase + case-insensitive
