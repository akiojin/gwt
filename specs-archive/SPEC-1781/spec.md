# AI ブランチ命名

## Background

gwt-tui の Wizard に AIBranchSuggest ステップを追加し、タスク説明テキストから AI がブランチ名候補を提案する機能を実装する。gwt-core の `ai::BranchNameSuggester` を使用して 3-5 個のブランチ名候補を生成する。

## User Stories

### US-1: タスク説明から AI がブランチ名を提案する

開発者として、Wizard でタスクの説明を入力すると AI が適切なブランチ名候補を 3-5 個提案し、その中から選択したい。

### US-2: 提案されたブランチ名を編集する

開発者として、AI が提案したブランチ名を選択後に微調整（編集）してから確定したい。

### US-3: AI 提案をスキップして手動入力する

開発者として、AI 提案をスキップして自分でブランチ名を直接入力する選択肢も持ちたい。

## Acceptance Scenarios

### AS-1: ブランチ名候補の生成

- Wizard の AIBranchSuggest ステップが表示される
- タスク説明を入力して送信
- 3-5 個のブランチ名候補がリスト表示される

### AS-2: 候補の選択と確定

- ブランチ名候補リストから矢印キーで選択
- Enter で確定し、次のステップに進む

### AS-3: 候補の編集

- ブランチ名候補を選択後、編集モードに入る
- ブランチ名を編集して確定

### AS-4: 手動入力へのフォールバック

- 「手動入力」オプションを選択
- テキストフィールドでブランチ名を直接入力

### AS-5: AI 接続エラー時のフォールバック

- AI API が利用不可の場合
- 手動入力モードに自動フォールバックし、エラーメッセージを表示

## Functional Requirements

| ID | 要件 |
|----|------|
| FR-001 | Wizard の AIBranchSuggest ステップとして実装する |
| FR-002 | タスク説明テキストを gwt-core の `ai::BranchNameSuggester` に送信する |
| FR-003 | 3-5 個のブランチ名候補を生成して表示する |
| FR-004 | ブランチ名候補の選択・確定をサポートする |
| FR-005 | 選択後のブランチ名編集をサポートする |
| FR-006 | 手動入力モードへのスキップをサポートする |
| FR-007 | ブランチ名のバリデーション（git ブランチ名として有効か）を実行する |

## Non-Functional Requirements

| ID | 要件 |
|----|------|
| NFR-001 | AI 候補生成は非同期で実行し、生成中はローディングインジケーターを表示する |
| NFR-002 | AI API タイムアウトは 10 秒とし、超過時は手動入力にフォールバックする |
| NFR-003 | 候補生成は 5 秒以内に完了する |

## Success Criteria

| ID | 基準 |
|----|------|
| SC-001 | タスク説明から 3-5 個のブランチ名候補が生成される |
| SC-002 | 候補の選択・編集・確定が動作する |
| SC-003 | 手動入力モードが機能する |
| SC-004 | AI エラー時にフォールバックが動作する |
| SC-005 | 生成されたブランチ名が git ブランチ名として有効 |
