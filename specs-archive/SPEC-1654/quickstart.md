# Quickstart

1. Launch gwt and confirm the first primary entry is `Branches`
2. Verify each branch row shows session count
3. Open a branch with no session and confirm launch flow starts
4. Open a branch with one session and confirm it enters that session directly
5. Open a branch with multiple sessions and confirm the selector appears
6. Start 4 or more sessions and confirm `equal grid` layout
7. Toggle maximize on the focused session, then return to grid
8. Open and close management tabs and confirm the previous session layout is restored
