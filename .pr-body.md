## Context

Launch Agent wizard caused main thread to block for 30-60+ seconds after pressing "Launch", freezing the entire UI. Root cause: `spawn_agent_window` ran synchronously on the tao event loop thread, blocking message delivery while performing heavy operations (git worktree resolution, Docker service checks, asset refresh). Solution divides work into two phases: immediate window creation with Starting status (fast, allows wizard close and UI update), followed by background async work with progress events streamed via WebSocket.

## Risk / Impact
